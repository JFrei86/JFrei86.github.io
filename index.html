Breakpoint


A Windows Store app designed for Windows Phone and Windows 8.1. Breakpoint is a native app that allows you to manage issues and milestones on GitHub. The app is written in C#.

RPI CSCI HW Server


The Computer Science Department uses a homework server which compiles and runs student code. This past year, I worked on improving the grading algorithms for this server with a team of four other students. The server is used by more than 500 students during a semester for Introduction to Data Structures and Introduction to Computer Science. The algorithm "modules" and the configuration system for this server are written in C and C++.

3

News Detection


Using Natural Language features and syntactic parsing of tweets from Twitter, I developed a supervised learning model that can determine whether a Twitter trend is considered newsworthy. My work with this is being used to assist research in generating summarized news articles from the detected tweets.
